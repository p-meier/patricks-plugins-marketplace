# Patrick's Plugins Marketplace

A plugin marketplace for Claude Code.

## Installation

```shell
/plugin marketplace add p-meier/patricks-plugins-marketplace
```

## Available Plugins

### deep-researcher

Deep research system with specialist agents, iterative analysis, and meta-principle extraction.

```shell
/plugin install deep-researcher@patricks-plugins
```

## Adding New Plugins

Add new plugin directories under `plugins/` and register them in `.claude-plugin/marketplace.json`.
